A professional WordPress plugin to delete users based on their roles.

== Description ==
The "Delete Users with Specific Roles" plugin allows administrators to delete users based on their assigned roles. It's a secure, user-friendly, and professional solution for WordPress site administrators.

= Features =
- Delete users by selecting a specific role.
- Easy-to-use interface in the WordPress admin dashboard.
- Add a "Settings" link in the plugins list for quick access.
- Professional code standards and translation-ready.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/delete-users-with-specific-roles` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to **Settings > Delete Users** to configure the plugin.

== Frequently Asked Questions ==

= Does this plugin delete users immediately? =
No, the deletion happens only after confirmation from the settings page.

= Is this plugin secure? =
Yes, the plugin uses WordPress nonces and sanitization functions to ensure security.

= Can I undo the deletion? =
Once users are deleted, they cannot be recovered unless you have a backup.
